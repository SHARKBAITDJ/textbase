Get a Free Phone Number with Google Voice

A real US number — free forever. Takes 2 minutes to set up.

Open Google Voice →
1
Go to voice.google.com
Sign in with any Google account. Click Get started.
2
Pick your number
Choose any US area code → search → pick a number you like. Yours forever, for free.
3
Verify with a real phone (one-time)
Google asks for a verification number just once. You can use your personal cell. You don't need to keep it linked.
4
Register it in TextBase
Go to Phone Numbers → Add Number → paste your Google Voice number (e.g. +15551234567) → select Google Voice as the carrier → Save.
How delivery works
TextBase emails {recipient's-10-digit-number}@txt.voice.google.com — Google's servers convert it to an SMS and deliver it to that phone. Free, no API key needed, no rate limits beyond your SMTP provider's daily cap.

API Reference

POST /api/v1/messages — Send SMS
{
  "to": "+15551234567",
  "carrier": "verizon",  // att|verizon|tmobile|sprint|boost|cricket|metro…
  "body": "Hello!",
  "from": "MyApp"   // optional sender label
}
GET /api/v1/messages — List messages
GET /api/v1/messages/{sid} — Get message status
POST /2010-04-01/Accounts/{AccountSid}/Messages.json — Twilio-compatible
Drop-in replacement for existing Twilio code. Pass Carrier=verizon as an extra form field for routing. Auth: HTTP Basic with AccountSID:AuthToken.
POST /webhook/inbound — Inbound SMS webhook
Point your SMS provider's inbound URL here. Accepts From, Body, To fields (Twilio format).
